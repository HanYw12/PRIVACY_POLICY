# PRIVACY_POLICY

# Política de Privacidad – Descarga SRI organizada

Esta extensión de Google Chrome no recopila, almacena ni comparte información personal de los usuarios.

## Datos que se manejan
La extensión únicamente utiliza datos presentes en la página web del Servicio de Rentas Internas (SRI) del Ecuador con el único fin de facilitar la descarga de comprobantes electrónicos solicitados explícitamente por el usuario.

## Almacenamiento
La extensión puede almacenar configuraciones básicas de forma local en el navegador del usuario utilizando la API de almacenamiento de Chrome. Esta información no se transmite a servidores externos.

## Descargas
La extensión utiliza el permiso de descargas únicamente para permitir que el usuario descargue archivos desde el sitio web del SRI.

## Terceros
La extensión no comparte datos con terceros ni utiliza servicios de análisis, seguimiento o publicidad.

## Cambios
Cualquier cambio en esta política será reflejado en esta misma página.
